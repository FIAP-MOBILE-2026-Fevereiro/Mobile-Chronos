# Design System — Void Protocol

## Paleta de Cores

| Token | Valor | Uso |
|---|---|---|
| `background` | `#0A0E1A` | Fundo principal (Void Black) |
| `accent` | `#0085FF` | Ion Blue — botões e CTAs |
| `cyan` | `#00D4FF` | Status online e latência |
| `gold` | `#FFB800` | Alertas e relógio lunar |
| `surface` | `#121926` | Fundo de cards sólidos |

## Tipografia

- **Títulos e displays:** `Syne-Bold` (700)
- **Corpo e labels:** `DMSans-Regular` (400), `DMSans-Medium` (500), `DMSans-Bold` (700)

## Espaçamentos

- Border radius padrão: `12px` (cards), `20px` (sub-cards), `30px` (botões pílula)
- Padding lateral: `16px`
- Padding superior (safe area): `50px`
